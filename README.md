# 🏢 Dorm_Sync - AI-Powered Hostel Management System

**Dorm_Sync** is a modern, full-stack, cross-platform mobile application designed to digitize traditional hostel operations. It replaces paper-based workflows with a smart, AI-driven digital ecosystem featuring real-time location tracking, NLP-based hazard detection, and predictive analytics.

---

## 🚀 Key Features

### 👨‍🎓 Student Portal
* **📍 Smart Attendance:** Two-factor presence verification using **Real-time GPS Geo-fencing** and **Face Recognition**.
* **🤖 AI Maintenance:** Students report issues, and an **NLP algorithm** scans for danger keywords (e.g., "Fire", "Spark", "Smoke") to instantly flag High-Priority alerts.
* **🎟️ Digital Gate Pass:** Students apply with exact date/time pickers and receive a digital QR-style slip upon warden approval.
* **🍲 Intelligent Mess Menu:** Expandable weekly accordion menu with a 5-star meal rating system.

### 👮‍♂️ Warden Dashboard (Web/Tablet Layout)
* **📊 Analytics Dashboard:** Real-time overview of students present, pending passes, and active complaints.
* **🔮 AI Food Waste Prediction:** A dynamic algorithm predicts the exact raw materials to cook (in kg/L) based on live student attendance, minimizing daily waste.
* **📢 Digital Notice Board:** Post urgent updates and attach images directly to the student feed via Multipart Form Data.
* **✅ One-Tap Approvals:** Instantly approve or reject gate passes from the sidebar.

---

## 🛠️ Tech Stack

* **Frontend:** Flutter (Dart) - Material 3 Modern UI, Geolocator, Image Picker
* **Backend:** Python (FastAPI), Python-Multipart
* **Database:** SQLite (Local, reliable, and privacy-focused)
* **AI/ML Logic:** Python (Natural Language Processing & Deterministic Predictive Algorithms)
